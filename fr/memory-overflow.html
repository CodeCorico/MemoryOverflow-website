Memory Overflow

Memory Overflow est un jeu de cartes TOTALLEMENT GRATUIT créé par des développeurs pour des développeurs.

Vous trouverez ici toutes les ressources pour pouvoir jouer avec d'autres développeurs et devenir LE Maître développeur.

Comment jouer ?

Memory Overflow est un jeu de cartes où deux joueurs s'affrontent pour le contrôle de La Machine. Le premier qui décode le code secret de La Machine gagne. Mais attention, si le code d'un joueur est plus grand que celui demandé par La Machine, elle le détruit instantanément. On appelle ça le Memory Overflow.
